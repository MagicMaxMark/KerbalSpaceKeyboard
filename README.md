# Kerbal Space Program Keyboard
Arduino code for a Kerbal Space Program Controller
### Pinout

```
             |Y   –––––  A0|
             |X   –––––  A1|
 Joystick    |Bt  ––––-  NC| Arduino
             |VCC ––––-  5v|
             |GND ––––- GND|
          
 Multiplexer |             | Arduino
```
